Transcription d’un manuscrit ancien – *L’Odyssée en vers burlesques*

Présentation :

Ce dépôt contient un travail de transcription réalisé à partir d’un manuscrit ancien, dans le cadre d’un projet utilisant la plateforme **eScriptorium** (FondueGD).  
L’objectif principal est de produire une transcription de pages numérisées, tout en documentant les choix méthodologiques et les difficultés rencontrées.

Le travail porte sur une œuvre burlesque inspirée de *L’Odyssée* d’Homère, attribuée à **Hugues de Picou**.

---

Description du document source :

- **Titre complet** : *L’Odyssée d’Homère, ou Les avantures d’Ulysse en vers burlesques*  
- **Auteur** : Hugues de Picou  
- **Type de document** : Livre imprimé ancien  
- **Langue** : Français (français classique, XVIIᵉ siècle)  
- **Nombre de pages** : 101 pages  
- **Institution de conservation** : Bibliothèque nationale de France (BnF)  
- **Source de la numérisation** : Gallica (BnF)

Les zones de ségmentation (titres, paragraphes, élément décoratifs) sont visibles par des espaces colorés dans l'interface eScriptorium (segmentation). Le type des zones n’est pas affiché directement sur l’image, mais il est bien défini dans les métadonnées de chaque région et visible lors de la sélection individuelle des zones.
